안드로이드 스튜디오를 설치한다.
메뉴창에서 File->open을 누른다.
과제파일 다운로드 받은 곳 찾아서 파일을 연다.(File name : test4)
초록색 안드로이드 이미지로 되어있는 test4파일을 열면 된다.
(*주의 할 점 : 과제를 다운로드 받은 위치경로에 한글이 들어가는 있는 곳이라면 과제파일이 열리지 않음. 경로에 한글이 없는 곳으로 과제파일을 저장해야한다.)

안드로이드 스튜디오 내에서 직접 안드로이드를 실행하기 위해서는 가상화 설정이 필요하다.
가상화 설정이 되어 있지 않다면 BIOS에 들어가 가상화 설정을 진행해준다.
가상화 설정이 되어있다면 Device Manager 버튼을 눌러 Create device를 누른다. 원하는 스펙의 안드로이드 휴대폰을 설정할 수 있다. (본인은 Pixel_3a_API-33_x86_64를 사용하였다.)
성공적으로 device를 만들었으면 Device에서 선택하여 Actions버튼을 눌러준다.
안드로이드 휴대폰이 뜨면 코드의 run 아이콘을 눌러 코드를 실행한다.
